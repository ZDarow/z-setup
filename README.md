# z-setup

Коллекция конфигураций, навыков, агентов и команд.

## Состав проекта

- **93 навыка** (skills/) — специализированные инструкции для разных задач
- **50 агентов** (agents/) — готовые агенты для различных сценариев
- **43 команды** (commands/) — часто используемые операции
- **Конфигурация** — opencode.json, tui.json, AGENTS.md
- **Документация** (docs/) — лучшие практики и примеры

## Установка

```bash
cd z-setup
bash install.sh
```

## Обновление

```bash
bash update.sh
```

## Удаление

```bash
bash uninstall.sh
```

## Структура

```
z-setup/
├── agents/          # Агенты
├── commands/        # Команды
├── skills/          # Навыки (93+)
├── docs/            # Документация
├── mcp/             # MCP-серверы
├── AGENTS.md        # Глобальные правила
├── opencode.json    # Основная конфигурация
├── tui.json         # Настройки интерфейса
└── install.sh       # Скрипт установки
```
